# SV Options Menu
Utilities for setting up an options menu in your Godot game. Currently
a work in progress.

## Installation
- SV Options Menu requires [Godot 4.6](https://godotengine.org/download/4.x)
- Clone the repository, then copy the `addons/sv_options_menu` folder
  into the addons folder of your own project.
- In future versions you will have to enable a plugin in order to
  automatically populate autoloads and configuration files.

## Usage
SV Options Menu is not usable in its current state. In future versions
there will be pre-made UI scenes you can place into your game to add
an options menu.
